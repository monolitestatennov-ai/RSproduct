# RSpace Services Integration

Платформа для риэлторов по интеграции цифровых сервисов и расчёту потенциального заработка.

## 🚀 Быстрый деплой

### Через GitHub Desktop (Рекомендуется)

1. Установите [GitHub Desktop](https://desktop.github.com/)
2. File → Add Local Repository → выберите эту папку
3. Нажмите "Publish repository"
4. Выберите организацию `monolitestatennov-ai` и название `RS-product`

### Через Vercel

1. Зайдите на [vercel.com](https://vercel.com)
2. Войдите через GitHub
3. Add New Project → выберите `monolitestatennov-ai/RS-product`
4. **Root Directory:** `client`
5. Deploy

## 📁 Структура проекта

```
client/          # React приложение
├── src/
│   ├── pages/   # Страницы (Home, Portfolio, PropertyCard, Dashboard)
│   ├── components/  # Компоненты (EarningsCalculator, Notifications, etc.)
│   └── ...
```

## 🛠️ Локальная разработка

```bash
cd client
npm install
npm run dev
```

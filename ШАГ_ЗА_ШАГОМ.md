# 🎯 ДЕПЛОЙ - ОЧЕНЬ ПРОСТАЯ ИНСТРУКЦИЯ

## Что мы делаем?
Мы загрузим ваш проект в интернет, чтобы его можно было открыть по ссылке в браузере.

---

## ШАГ 1: Откройте терминал на вашем Mac

1. Нажмите на иконку поиска (лупа) в правом верхнем углу
2. Наберите: **Terminal**
3. Нажмите Enter
4. Откроется черное окошко - это терминал

---

## ШАГ 2: Перейдите в папку проекта

В терминале скопируйте и вставьте эту команду (нажмите Enter):

```bash
cd "/Users/alekseykozlov/Desktop/Rs Product"
```

**Что это значит?** Мы говорим компьютеру: "Иди в папку с нашим проектом"

---

## ШАГ 3: Создайте репозиторий на GitHub

1. Откройте браузер
2. Зайдите на сайт: **github.com**
3. Войдите в свой аккаунт (или зарегистрируйтесь)
4. Нажмите на **зеленую кнопку "New"** (или плюсик в правом верхнем углу)
5. Напишите название: **rspace-project** (или любое другое)
6. НЕ ставьте галочки (не добавляйте README, .gitignore, лицензию)
7. Нажмите зеленую кнопку **"Create repository"**

**Запомните:** Какой у вас username на GitHub? (напишите его, он понадобится)

---

## ШАГ 4: Подготовьте проект к загрузке

Вернитесь в терминал и выполните команды по очереди (каждую нажимайте Enter):

**Команда 1:**
```bash
git init
```
(Это как сказать: "Начни отслеживать изменения в этой папке")

**Команда 2:**
```bash
git add .
```
(Это как сказать: "Добавь все файлы в список для отправки")

**Команда 3:**
```bash
git commit -m "Первый коммит"
```
(Это как сказать: "Сохрани все файлы")

---

## ШАГ 5: Подключите проект к GitHub

В терминале выполните (замените YOUR_USERNAME на ваш реальный username с GitHub):

```bash
git remote add origin https://github.com/YOUR_USERNAME/rspace-project.git
```

**Пример:** Если ваш username `aleksey123`, то команда будет:
```bash
git remote add origin https://github.com/aleksey123/rspace-project.git
```

**Важно:** Название репозитория должно совпадать с тем, что вы создали на GitHub!

---

## ШАГ 6: Отправьте код на GitHub

Выполните две команды:

**Команда 1:**
```bash
git branch -M main
```

**Команда 2:**
```bash
git push -u origin main
```

**Что произойдет:** Ваш код загрузится на GitHub. Это может занять 1-2 минуты.

---

## ШАГ 7: Задеплойте на Vercel (чтобы сайт был в интернете)

1. Откройте новый сайт: **vercel.com**
2. Нажмите **"Sign Up"** (Регистрация)
3. Выберите **"Continue with GitHub"** (Войти через GitHub)
4. Разрешите доступ
5. Нажмите **"Add New Project"** (Добавить новый проект)
6. Выберите ваш репозиторий **rspace-project**
7. В настройках найдите **"Root Directory"** и напишите: **client**
8. Нажмите **"Deploy"** (Развернуть)

**Ждите 1-2 минуты...**

9. Когда увидите "Congratulations!" - нажмите на ссылку
10. **ГОТОВО!** Ваш сайт теперь в интернете! 🎉

---

## Если что-то пошло не так:

### Ошибка "repository not found"?
- Проверьте, что username правильный
- Проверьте, что название репозитория совпадает

### Ошибка "permission denied"?
- Проверьте, что токен правильный
- Попробуйте создать новый токен на GitHub

### Не работает деплой на Vercel?
- Убедитесь, что указали Root Directory: **client**
- Проверьте логи в Vercel (там будет написано, что не так)

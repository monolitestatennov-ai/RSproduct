### Страховка ипотеки (PropertyCard.tsx)

**Формула расчёта:**
```
Стоимость страховки в год = Цена квартиры × 0.01 (1%)
Доход в год = Стоимость страховки × 0.3 (30% маржа)
Доход за 12 лет = Доход в год × 12
```

**Пример для квартиры 5.8M ₽:**
```
Страховка в год = 5,800,000 × 0.01 = 58,000 ₽
Доход в год = 58,000 × 0.3 = 17,400 ₽
Доход за 12 лет = 17,400 × 12 = 208,800 ₽

Для разных количеств клиентов:
- 10 клиентов: 17,400 × 10 = 174,000 ₽/год → 2,088,000 ₽ за 12 лет
- 50 клиентов: 17,400 × 50 = 870,000 ₽/год → 10,440,000 ₽ за 12 лет
- 100 клиентов: 17,400 × 100 = 1,740,000 ₽/год → 20,880,000 ₽ за 12 лет
```

## 💰 Расчёты заработка

#### Сервисы


### Для всего портфеля (Portfolio.tsx)

**При условии, что все 12 объектов проданы с использованием всех сервисов:**

#### Расчёт по сервисам
```
Средняя маржа на объект = 80K ₽
Всего объектов = 12
Общая маржа = 80K ₽ × 12 = 960K ₽
```

#### Расчёт по страховке
```
Средняя цена объекта = 10.8M ₽
Страховка в год на объект = 10.8M × 0.01 = 108K ₽
Доход в год на объект = 108K × 0.3 = 32.4K ₽
Общий доход в год = 32.4K × 12 = 388.8K ₽
Общий доход за 12 лет = 388.8K × 12 = 4,665.6K ₽ (≈ 4.7M ₽)
```

#### Итоговый потенциальный заработок
```
От сервисов: 960K ₽
От страховки (1 год): 388.8K ₽
От страховки (12 лет): 4,665.6K ₽
---
Итого в первый год: 960K ₽ + 388.8K ₽ = 1,348.8K ₽ (≈ 1.35M ₽)
Итого за 12 лет: 960K ₽ + 4,665.6K ₽ = 5,625.6K ₽ (≈ 5.6M ₽)
```

## 📈 Ипотечный калькулятор

### Формула расчёта ежемесячного платежа

```
Ежемесячный платёж = (Сумма × Месячная ставка × (1 + Месячная ставка)^Месяцы) / 
                     ((1 + Месячная ставка)^Месяцы - 1)

Где:
- Сумма = Цена квартиры
- Месячная ставка = Годовая ставка / 100 / 12
- Месяцы = 360 (30 лет)
```

### Пример для квартиры 5.8M ₽ при ставке 13.99%

```
Месячная ставка = 13.99 / 100 / 12 = 0.011658
Месяцы = 360

Ежемесячный платёж = (5,800,000 × 0.011658 × (1.011658)^360) / 
                     ((1.011658)^360 - 1)
                   ≈ 69,000 ₽
```

## 🔄 Способы оплаты

- Клиент получает счёт на email
- Оплата через банк или платёжную систему
- Сроки: 5-10 рабочих дней

### Оплатить картой
- Прямая оплата в системе
- Сроки: мгновенно
- Комиссия: 2-3% (зависит от банка)

## 📝 Примечания по данным

1. **Цены сервисов** - примерные и могут быть изменены в зависимости от условий
2. **Маржа от страховки** - установлена на 30%, может быть пересмотрена
3. **Срок ипотеки** - стандартный 12 лет (360 месяцев), может варьироваться
4. **Ставки по ипотеке** - 13.99% - примерная текущая ставка, обновляется в реальном времени
5. **Объекты недвижимости** - тестовые данные, в production будут реальные объекты из БД

---

## 🔐 Безопасность данных

- Все расчёты выполняются на клиенте (браузер)
- Никакие финансовые данные не передаются на сервер
- Данные объектов захардкодены (в production будут из БД)
- Нет сохранения личных данных клиентов

## 🚀 Как обновить данные

### Изменить цены сервисов
1. Откройте `client/src/pages/Home.tsx`
2. Найдите `const allServices = [`
3. Измените `price:` и `margin:` для нужного сервиса
4. Сохраните файл - изменения применятся автоматически

### Добавить новый объект в портфель
1. Откройте `client/src/pages/Portfolio.tsx`
2. Найдите `const allProperties = [`
3. Добавьте новый объект в конец массива
4. Сохраните файл

### Изменить расчёт страховки
1. Откройте `client/src/pages/PropertyCard.tsx`
2. Найдите `const insurancePerYear = Math.round(property.price * 0.01);`
3. Измените коэффициент (0.01 = 1%)
4. Сохраните файл

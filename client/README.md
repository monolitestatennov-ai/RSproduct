# RSpace Services Integration

Платформа для риэлторов по интеграции цифровых сервисов и расчёту потенциального заработка.

## 🚀 Быстрый старт

### Установка зависимостей

```bash
pnpm install
```

### Запуск dev сервера

```bash
pnpm dev
```

Откройте в браузере: `http://localhost:3000`

### Сборка для production

```bash
pnpm build
```

## 📁 Структура проекта

```
client/
├── public/
```

- **React 19** - UI библиотека

- **Wouter** - Маршрутизация
- **shadcn/ui** - UI компоненты
- **lucide-react** - Иконки

## 📄 Страницы

### Home.tsx - Создание
- Выбор сервисов из 5 основных
- Расчёт маржи и бонусов в реальном времени
- Выбор способа оплаты
- Боковая панель с итоговой суммой

### PropertyCard.tsx - Опубликована
- Информация об объекте
- Доступные сервисы с УТП
- Расчёт пассивного дохода от страховки
- Ипотечный калькулятор

### Portfolio.tsx - Портфель
- Сетка из 12 объектов недвижимости
- Фильтрация по типу
- Сортировка по цене и площади
- Модальные окна для услуг и поделиться
